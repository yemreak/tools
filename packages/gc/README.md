# @yemreak/gc

Git commits with AI-generated messages using Claude Code CLI

## Why

Clear commit messages in seconds, not minutes

## Install
```bash
bun add -g @yemreak/gc
```

## Usage
```bash
git add .
gc
```

## Environment
Requires Claude Code CLI installed:
```bash
# Check if installed
which claude

# Expected location
~/.local/bin/claude
```
