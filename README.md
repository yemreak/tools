# tools

## Install

```bash
# npm (recommended)
npm install -g @yemreak/fetch

# bun
bun add -g @yemreak/fetch

# yarn
yarn global add @yemreak/fetch
```

## Tools

| Command | NPM Package |
|---------|-------------|
| `yfetch` | `@yemreak/fetch` |

## Usage

```bash
yfetch "https://developer.apple.com/documentation/activitykit"
```
